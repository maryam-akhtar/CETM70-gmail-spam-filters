# Gmail Spam Filters – CETM70 Poster Project
 
The poster explores how computer science techniques such as machine learning, neural networks, and natural language processing (NLP) have been applied to tackle the issue of **spam filtering in Gmail**.

## Key Focus
- How Gmail uses **AI-driven filters** to detect spam and phishing emails  
- The role of **neural networks** and **optical character recognition (OCR)**  
- Consideration of **legal, ethical, social, and professional** issues surrounding data protection and user privacy

## Tools & Methods
- Literature review and data analysis  
- Visual representation of key findings in poster format  
- Harvard referencing of all research sources

## Summary of Findings
- Gmail’s spam filter uses a combination of **rule-based**, **machine learning**, and **neural network** methods  
- Effectiveness: **~99% spam detection rate** with **1% false positives**  
- Demonstrated via a 7-day data sample showing high accuracy and user protection
